# Keyword 분석 절차 설계

1. 공유 디렉토리(/share/)에 있는 상담 데이터(STT)를 불러와서 전처리 수행

    - 전처리
    - 명사 추출
    - 빈도수
    - 빈도수가 낮은 것들(한번 나온것들/한글자로 이루어진 토큰)을 불용어 처리(리스트에 추가)
    - 원본 파일 / 토큰으로 구성된 파일(전처리된 파일) / 단어 / 불용어 리스트를 각각 파일로 저장

2. 불용어 제거

    - 전단계에서 구성된 불용어 리스트를 활용해서 명사단위로 자른 문장들에서 불용어를 제거

3. 각 단어별 TF-IDF 값을 구한다.

    - 상담데이터 문서에 나온 단어에 대한 TF 값과 DF 값을 바탕으로 각 단어별로 TF-IDF 값을 구하고 그 값들을 별도의 파알로 저장

4. 단어에 대한 가중치 부여

    - 어떤 단어들이 동시에 출현했을 경우 가중치를 부여
    - 단어가 동시에 출현했다고 하더라도 각 단어의 빈도에 따라 빈도가 높은 단어에 더 높은 가중치를 부여
    - 값을 Normalize 하기 위해 IDF 값을 곱해 다른 문서에서 잘 나오지 않는 단어의 가중치를 높여줌.

5. 4단계까지 진행해서 만들어진 단어사전을 이용해 키워드 추출

    - 각 상담 데이터 파일 별로 가중치가 높은 키워드를 추출
