1. 업무진행형황

    - 업무회의: SKT 주간회의 참석(을지로 SKT타워)

        --> 참석: SKT(현업담당 3명), SKCNC(신현오BJ 외4명), Nextobe(이태희)

        --> 회의 내용

            == 현재까지의 POC진행사항 중간보고

                ==> TA(Text Analysis) 업무에 대한 비즈니스적인 접근
	        ==> VOC/STT 변환 업무 흐름 및 STT분석 업무 설명 및 시연
                ==> TO-BE 에 대한 시스템 흐름 및 인프라 투자 관련 설명

            == 보고 내역 토의
			
		(SKT 고객사 의견)
                ==> TA 업무에 대한 End-Feature 가 보이지 않음.
                ==> Keyword 추출 및 Keyword Set에 대해서는 감성분석이 필요함
                    ex) 안심 옵션 해지 >> 해지를 하는 것인지.. 해지를 하지 않는 것인지 판단이 되지 않음.
                ==> Keyword 가 아닌 Key Sentence 가 필요함

        --> 회의 결과

            == 현재 구현된 Keyword/Keyword Set 으로는 문서(VOC)의 주제가 보이지 않음.
 
            == TA 의 End-Feature 제시 필요

            == Keyword >> Key Sentence
